Bomb Crypto Bot: Auto clicker para jogo NFT em navegador, desenvolvido com pyautogui.
O bot foi criado para logar em sua conta e colocar os heróis do jogo para trabalhar regularmente, para rodar 24/7, ele possuí mecanismos para lidar com erros eventuais.
