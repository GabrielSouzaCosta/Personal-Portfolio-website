ExcelFiller foi criado com a intenção de agilizar na criação de planilhas que usam muitos dados repetidos...
É possível cadastrar textos frequentemente usados e seleciona-lós rapidamente posteriormente...
Além de contar com diversos outros tipos de dados, por exemplo data automática e textos personalizados...
O website tem domínio próprio podendo ser acessado pelo endereço www.excelfiller.com. O backend está hospedado no Heroku.
Este projeto tem como objetivo resolver um problema bastante específico, buscando evitar a necessidade de excessivos ctrl + c | ctrl + v na criação de planilhas...
